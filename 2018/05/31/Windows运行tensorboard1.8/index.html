Windows 运行 tensorboard 1.6

本篇讲述一下 Windows 环境下如何运行 Tensorboard 可视化 graph.

环境

  • Windows 7
  • tensorflow 1.6/1.8

tensorlfow 1.6 在 windows 上通过 pip3 install tensorflow 安装，同时会顺带安装 tensorboard 工具，1.6 版本已经修改过了文件路径，具体的使用方式，参看如下例子。本机是1.8版本，1.6 也同样适用。
